---- **Ahorcado Pokemon**------

Este proyecto es un programa desarrollado en Java para poder jugar el tipico juego del ahorcado. 
Fue desarrollado en equipo y permite llevar el control de jugadores, puntajes y interfaz grafica.

**Funcionalidades**

- Generación de jugadores
- Control de juegos perdidos y ganados
- Visualización del puntaje de los jugadores
- Interfaz grafica agradable 

**Requisitos**
Para poder jugar se deben crear inicialmente los jugadores ya que no maneja base de datos los juagdores se crean atravez de una interfaz
en la cual se asigna su saldo (puntaje) y se pueden eliminar jugadores. 
al comenzar el juego se permiten resolver tres ahorcados, al final de hace un recuento de palabras encontradas y el puntaje actual.

**Uso**
Ejecuta el programa desde el archivo  en src: inicio
